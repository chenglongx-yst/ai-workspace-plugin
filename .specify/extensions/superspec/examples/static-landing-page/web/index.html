SuperSpec

Specification-driven development for modern teams

Get Started

Core Commands

status

Check the current state of your specification and implementation progress

brainstorm

Generate creative ideas and explore solution spaces for your feature

tasks

Decompose your plan into concrete, actionable implementation tasks

execute

Run the implementation workflow, processing tasks in order

review

Verify the implementation against the specification and constitution
